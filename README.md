## 安装依赖项 Project setup

```
npm install
```

### 开发环境：编译并热加载 Compiles and hot-reloads for development

```
npm run serve
```

### 生产环境：编译并压缩 Compiles and minifies for production

```
npm run build
```

### 单元测试 Run your unit tests

```
npm run test:unit
```

### 风格修正 Lints and fixes files

```
npm run lint
```
